### Descritpion

Braile beacon system is a hardware solution made to help (no vidente) people to find braile information.
It focuses maily on museums or public places where there is braile, but not the ground paths for people with visual impairement.

The system consists of a tag and a receiver. The tag is located where the braile is, 
the receiver is a haptic device that the user weres, and the user can use the tag to find the braile beacons.
Other functions of the beacons is to help the user locate doors, stairs or main locations in the building. 

It is not meant to replace braile or the specialized paths, It only helps the user to find those things above.
The system is an assistive location cue, not certified navigation, obstacle detection, or a replacement for a cane, guide dog, mobility training, or proper tactile signage.
